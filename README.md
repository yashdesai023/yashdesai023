## Featured Projects

### vectorDBpipe — Vector Search Framework
[![PyPI](https://img.shields.io/pypi/v/vectordbpipe?style=flat-square&color=3775A9)](https://pypi.org/project/vectordbpipe/)
[![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)](https://github.com/yashdesai023/vectordbpipe-backend)

Open-source PyPI package that unifies FAISS, Pinecone, and ChromaDB into a single vector-search framework for LLM agents. Implements Omni-RAG, GraphRAG, and a Vectorless Engine. Ships as a real pip-installable package with a FastAPI dashboard.

**Benchmarks:** 18.4ms p50 latency · 0.92 MRR · 42.5 chunks/s ingestion

`pip install vectordbpipe`

[Backend Repo](https://github.com/yashdesai023/vectordbpipe-backend) · [Frontend Repo](https://github.com/yashdesai023/vectordbpipe-frontend) · [PyPI](https://pypi.org/project/vectordbpipe/)
